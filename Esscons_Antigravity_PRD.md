**Project:** Esscons Construction Machinery Supplier Website Redesign  
**Platform:** Antigravity  
**Date:** January 31, 2026  
**Company Focus:** Construction Equipment Supplier, Cement & Mining Industries

## 1. EXECUTIVE SUMMARY

Rebuild the Esscons website (esscons.com) using Antigravity with modern animations, interactive product showcases, and an animated road roller asset. The redesigned website will enhance brand presence, improve user engagement, and showcase products with immersive animations aligned with the construction equipment industry.

## 7. DESIGN & ANIMATION SPECIFICATIONS

**Secondary Colors:**

---

### 7.2 Typography

**Headings:** Bold, modern sans-serif (Google Fonts: Poppins or Inter)
- H1: 48px bold
- H2: 36px bold
- H3: 24px semi-bold
- H4: 20px semi-bold

**Body Text:** Clean sans-serif (Google Fonts: Roboto or Open Sans)
- Regular: 16px
- Small: 14px
- Line height: 1.6

**Accent/Tech Text:** Monospace for specs (IBM Plex Mono)

---

### 7.3 CORE ANIMATION REQUIREMENTS

#### 7.3.1 Road Roller Animated Hero (PRIMARY FEATURE)

**Animation Characteristics:**
- **Movement Pattern:** Antigravity road roller moving across a terrain
- **Path:** Diagonal or S-curve across the hero section
- **Speed:** 6-8 second loop
- **Effects:**
  - Subtle rotation/tilting as it moves
  - Compaction particle effects (dust/debris)
  - Vibration effect on the ground beneath
  - Shadow that follows movement
  - Terrain deformation/visual effect
  - Glow effect on wheels/contact points

**Trigger:** Auto-play on page load, can be configured to pause/resume on user interaction

**Technical:**
- WebGL/Canvas-based animation
- GPU-accelerated
- Responsive scaling for different screen sizes
- Fallback static image for low-performance devices

**Customization Options in Antigravity:**
- Speed slider
- Color overlay/theme options
- Particle density
- Loop/single-play option
- Sound effects (optional - subtle machinery sounds)

---

#### 7.3.2 Scroll-Triggered Animations

**Elements with Scroll Animations:**
- Cards fade in on scroll
- Counters animate from 0 to final number
- Text reveals with staggered timing
- Images scale up on viewport entry
- Backgrounds parallax shift
- Testimonials slide in
- Icons rotate into view

**Animation Timing:**
- Entrance animations: 0.6-0.8 seconds
- Stagger between elements: 0.1-0.15 seconds
- Easing: ease-out for smooth feel

---

#### 7.3.3 Interactive Hover Effects

**Product Cards:**
- Scale: 1 → 1.05 on hover
- Shadow increase
- Content overlay fade in
- CTA button color change

**Navigation Items:**
- Underline animation (width: 0 → 100%)
- Text color transition
- Background color subtle shift

**Buttons:**
- Background color change
- Text color change
- Icon animation (arrow slides right)
- Ripple effect on click

**Images:**
- Brightness increase
- Slight zoom
- Overlay color fade

---

#### 7.3.4 Transition Animations

**Page Transitions:**
- Fade out → Fade in (0.3 seconds)
- Smooth scroll to top on route change

**Modal/Dialog:**
- Backdrop fade in
- Modal scale: 0.8 → 1 with fade
- Exit: reverse animation

**Carousel/Slider:**
- Testimonials: Auto slide every 5 seconds
- Smooth fade or slide transitions
- Navigation dots show progress

---

### 7.4 Responsive Design Breakpoints

- **Desktop:** 1920px and above (full animations)
- **Laptop:** 1440px (full animations)
- **Tablet:** 768px-1024px (slightly reduced animation complexity)
- **Mobile:** 320px-768px (simplified animations, hero slider instead of full animation)

## 8. TECHNICAL SPECIFICATIONS

### 8.1 Antigravity Implementation

**Core Technologies:**
- **Framework:** Antigravity (next-gen web framework)
- **Animation Library:** Built-in Antigravity animations + Three.js for 3D/WebGL
- **State Management:** Antigravity state system
- **CMS Integration:** Headless CMS (optional - for blog/content management)

### 8.2 Performance Targets

- **Page Load Time:** < 2 seconds (Core Web Vitals)
- **Largest Contentful Paint (LCP):** < 2.5 seconds
- **First Input Delay (FID):** < 100ms
- **Cumulative Layout Shift (CLS):** < 0.1

### 8.3 SEO Optimization

- **Meta Tags:** Title, description, keywords for each page
- **Structured Data:** Schema.org markup for Organization, Product, LocalBusiness
- **Sitemap:** XML sitemap with all pages
- **Robots.txt:** Proper indexing rules
- **Open Graph Tags:** For social media sharing
- **Mobile-First Indexing:** Responsive design priority

### 8.4 Backend Requirements

**Contact Form Backend:**
- Form validation (client-side + server-side)
- Email notification to Esscons team
- Lead capture in CRM (if integrated)
- Automated thank-you email to user

**Optional Features:**
- Blog management system (if blog added)
- Quote/inquiry request tracking
- User authentication for customer portal
